    Como configurar

    Use a Admin API com header Authorization: Bearer <sua-api-key>. Na primeira execução, a chave bootstrap é gerada automaticamente e impressa nos logs do container (ou defina ADMIN_BOOTSTRAP_KEY).

    PUT /api/admin/config/Forwarding
{ "host": "www.exemplo.com", "domains": ["exemplo.com", "*.exemplo.com"] }

PUT /api/admin/config/MaxSessions
10

PUT /api/admin/config/JsBridge
{ "enable": true }

    Documentação OpenAPI (requer auth): /openapi/v1.json
